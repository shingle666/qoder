# 项目设置

针对不同项目类型的最佳配置建议。

## Web 前端项目
- 支持 React、Vue、Angular
- 配置 ESLint 和 Prettier
- 启用组件智能提示

## 后端 API 项目
- 配置数据库连接
- 设置路由分析
- 启用 API 文档生成

## 移动应用项目
- React Native 配置
- Flutter 支持设置
- 设备特定优化

## 配置示例
```json
{
  "projectType": "web-frontend",
  "framework": "react",
  "language": "typescript",
  "testFramework": "jest"
}
```
